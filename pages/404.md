你来到这个页面，通常有两个原因。

## 一、链接错误

原因：博客搬迁造成的旧链接失效。

例子：《改进博客园Markdown显示功能(加代码行号、显示代码所用编程语言)》的链接是

```
https://enjoy233.cn/2019/03/03/cnblogs-markdown-code-opt
```

请修改成

```
https://web.geekzl.com/2019/03/03/cnblogs-markdown-code-opt
```

改动要点：将域名 `enjoy233.cn` 改成 `web.geekzl.com`。

**备用办法：** 在「[分类](/categories/)」中找。

## 二、维度攻击

不明力量入侵，你想要访问的页面被摧毁了。
